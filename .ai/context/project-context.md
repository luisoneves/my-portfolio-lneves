## Como usar

Este arquivo é o "adaptador" do `.ai` para o seu projeto específico.

Preencha as seções relevantes. Deixe em branco o que não se aplica.  
Atualize quando o projeto evoluir — contexto desatualizado é pior que ausência de contexto.

Para agentes de IA: este arquivo é leitura obrigatória antes de qualquer operação no projeto.
